Windows XP

Data: 25 de Outubro de 2001

Veja mais em: Wikipédia - Windows XP

O Windows XP é uma família de sistemas operacionais de 32 e 64 bits produzido pela Microsoft, para uso em computadores pessoais, incluindo computadores residenciais e de escritório, notebooks, tablets e media centers.

O nome "XP" deriva de eXPerience.

Durante o seu desenvolvimento, o projeto tinha o nome de código "Whistler", em homenagem a cidade de Whistler no Canadá, uma vez que lá se localiza o resort Whistler Blackcomb onde alguns dos desenvolvedores se hospedaram.
